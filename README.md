# Car_Price_Prediction_System
This car price prediction project  predicts the selling price of used cars using Machine Learning. A Random Forest Regressor model is trained on a car dataset with features like year, fuel type, transmission, and kilometers driven. The model is deployed using Streamlit to provide an interactive web app for price prediction.
